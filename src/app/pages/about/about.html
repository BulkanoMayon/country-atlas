CityExplore / Country Atlas

About the Project

This Angular-based travel and lifestyle guide was developed to help users explore destinations through country information, regional browsing, destination highlights, map access, and saved favorites. The project demonstrates routing, API integration, browser-based authentication, and responsive design in a front-end-only environment.

What this project offers

Destination Exploration

Users can browse destinations through country cards, discover basic facts, and view detailed pages for each location.

Travel & Lifestyle Information

The details pages present attractions, popular cities, food, culture, events, and suggested activities for a more lifestyle-oriented experience.

Interactive Navigation

The app uses Angular routing, search filters, regional browsing, and direct map links to help users move through travel content smoothly.

Simulated User Features

Login, registration, and favorites are handled through browser storage to simulate account and personalization features without a backend.

Project Scope

The application is built as a front-end-only travel and lifestyle guide using Angular, HTML, and CSS. It focuses on destination discovery, interactive browsing, user-friendly design, and API-powered data presentation while remaining within browser-based limitations.

Frequently Asked Questions

Need more help?

What kind of system is this project?

It is a front-end travel and lifestyle exploration app built with Angular. It combines destination information, map access, search tools, and user features in one interface.

How is lifestyle content shown?

The country details pages include destination highlights such as attractions, cities, food, cultural elements, events, and things to do.

Does the project use a backend?

No. This version is fully front-end only. User account data and favorites are stored in the browser through localStorage.

Can users explore destinations by region?

Yes. Users can search and filter by major world regions such as Asia, Europe, Africa, Americas, and Oceania.

Does the project include map support?

Yes. Each destination page includes a direct map link so users can locate and view the selected destination externally.
