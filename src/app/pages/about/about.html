Country Atlas

About Country Atlas

Country Atlas is an Angular-based country explorer app that helps users browse countries, search by name, view detailed country information, explore regions, and save favorite countries in one clean interface.

Go to FAQ Contact Us

What this app does

Explore Countries

Browse countries in a clean grid layout with flags, capitals, populations, and currencies.

Search and Filter

Search countries by name and filter by regions like Asia, Europe, Africa, Americas, and Oceania.

Country Details

View richer information such as native name, languages, time zones, maps, and highlights.

Favorites

Save countries you like and revisit them later through your favorites page.

Frequently Asked Questions

Need more help?

What API does Country Atlas use?

The app uses the REST Countries API for country data and a fallback description system for extra country information.

Do I need an account to browse countries?

No. You can explore countries without an account, but creating an account lets you use features like favorites.

How does favorites work?

Favorites are saved locally in your browser using localStorage, so they stay available on the same device and browser.

Why are some country descriptions short?

Some descriptions use a fallback summary when external description data is limited or unavailable.

Can I browse by region?

Yes. You can use the Search page to filter countries by Asia, Europe, Africa, Americas, and Oceania.
